# personal medical record

Developed a comprehensive Medical Record app designed to manage and store patient health information securely. The app allows users to input, update, and retrieve medical records efficiently. Key features include patient data management, secure access controls, and integration with healthcare providers for real-time updates. The project demonstrated skills in data management, user interface design, and healthcare IT standards.
